# Beta Bank: Customer Churn Prediction

## About
Beta Bank is losing customers at a steady rate. To improve retention, this project involves building a classification model to predict whether a customer will leave the bank soon. The goal is to achieve the maximum possible **F1 score** (minimum threshold of **0.59**) to balance precision and recall effectively.

## Technical Highlights
· **Data Preparation**: Handled imbalanced classes using strategies like class weighting and resampling.

· **Feature Engineering**: Applied **One-Hot Encoding (OHE)** to categorical variables and ensured feature alignment.

· **Algorithms**: Trained and compared **Logistic Regression**, **Random Forest**, and **Gradient Boosting** models.

· **Optimization**: Utilized **GridSearchCV** for hyperparameter tuning to maximize model performance.

· **Metrics**: Tracked **F1-score** and **AUC-ROC** to evaluate model reliability on imbalanced data.
